# Airplane Booking System ✈️

A Python-based application for managing domestic airplane flight reservations, automated passenger baggage handling, and data-driven fee calculations. 

## Features
- **Flight Management:** Scheduling and tracking domestic flights.
- **Passenger Info:** Handling passenger details and bookings.
- **Baggage & Food:** Automated logic for calculating baggage fees and food management.
- **Data Driven Logic:** Utilizes flight schedule datasets (CSV) for accurate processing.

## Project Structure
- `Airplane/`: Core flight booking classes.
- `airline_baggage/`: Modules for handling baggage policies and fees.
- `class_flights/`: Flight classification and categorization.
- `passenger_info/`: Passenger data management.
- `food.py`: In-flight food logic.
- `ProjectCompro.ipynb`: Jupyter Notebook demonstrating the project logic and execution.
- `thailand_flights_updated_schedule...csv`: Dataset containing flight schedules.

## Technologies Used
- Python 3
- Jupyter Notebook
- Pandas (for CSV parsing)

## Getting Started
To view the main demonstration of the project, open the Jupyter Notebook:
```bash
jupyter notebook ProjectCompro.ipynb
```
